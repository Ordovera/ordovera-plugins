# AGENTS.md

Small React 18 application for internal dashboards.

## Tech Stack

- React 18
- TypeScript
- Vite

## Commands

- `npm run dev`
- `npm run test:api`
- `npm run lint`

## Project Structure

- `src/lib/auth/` contains the auth middleware
- `src/components/` contains reusable UI pieces

## Code Standards

- Use Vitest for all tests
- Prefer ESM imports

## Do NOT

- Do not change auth behavior without review
